# Dark Horse - Quick Reference

## Game Info
- **Players:** 2-6 | **Age:** 8+ | **Time:** 15min
- Horse racing game with strategy and luck

## Components
- **7 Horse Cards** (numbered 1-7)
- **14 Betting Cards** (2 per horse)
- **33 Action Cards** (movement cards)
- **3 Dark Horse Tokens** (+5/-3 points)

## Objective
Use action cards to get your horses to higher ranks. Highest score wins.

## Setup

### 1. Card Distribution
- **Betting:** 2 cards each (3 for 2p), keep secret, face down
- **Action:** 8/7/6/6/5 cards (for 2/3/4/5/6 players)

### 2. Horse Starting Order
- Players take turns placing horse cards left or right of existing cards
- Cannot insert between cards
- Direction: ← [Horses] → Front

### 3. Dark Horse
- Flip 7th (last) horse card = Dark Horse
- Place tokens: 1/2/2/3/3 (for 2/3/4/5/6 players)

## Gameplay

### Turn Order (Clockwise)

1. **Take Dark Horse Token** (Optional)
   - Max 1 per player
   - Cannot take with only 1 card in hand

2. **Play Action Card** (Mandatory)
   - Place face up in front of you

3. **Execute Card** (Mandatory)
   - Move horses by switching positions

### Movement Rules

- **Move forward N:** Pass N horses, switch with Nth horse
- **Passed horses:** Each drops 1 rank
- **Multi-horse cards:** Both move same direction
- **Edge cases:** 1st rank can't go forward → goes backward

### Special Cards

- **Rider Fall Off:** 3rd rank → 7th rank
- **Exchange Betting:** You or another player exchanges 1 betting card

### Important

- Must play card even if disadvantageous
- Must execute card effect (no skipping)
- Game ends when all cards played

## Game End

### When Game Ends
All action cards played → Game ends

### Final Ranking
```
← [7th][6th][5th][4th][3rd][2nd][1st] → Front
```
Rightmost = 1st, Leftmost = 7th

## Scoring

1. **Reveal betting cards**
2. **Base score:** Sum points from your betting cards (check scoring table on card back)
3. **Double bonus:** 2 identical betting cards = double points for that horse
4. **Dark Horse token:**
   - Dark Horse in top 3 (1st/2nd/3rd): **+5 points**
   - Dark Horse in bottom 4 (4th/5th/6th/7th): **-3 points**
5. **Winner:** Highest total score

### Remember
- Dark Horse = 7th position horse at game start
- Betting cards: 2 per player (3 for 2p)
